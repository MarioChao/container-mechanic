# Changelogs

## [v0.0.0] Container mechanic class | 2026/05/13

Created the `ContainerMechanic` class module.
- Constructor: `.new(tagName)`.
- Setter functions:
    - `:setFunction_getConfigFromInstance(newFunction)`.
    - `:setFunction_setUpContainer(newFunction)`.
    - `:setFunction_cleanUpContainer(newFunction)`.
- Run function: `:run()`.

Created test place scripts with an example bouncing pad mechanic.

Structure inspired by that used in [ContainerEvent](https://github.com/MarioChao/container-event) v0.0.0.
